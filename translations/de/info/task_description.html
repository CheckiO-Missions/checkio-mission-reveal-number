Das Ziel ist es, die Zahl aus der Zeichenkette zu "extrahieren": Verketten Sie nur die Ziffern, das Zahlenzeichen und "." (falls vorhanden) und geben Sie das Ergebnis im Zahlenformat zurück. Wenn keine Zahl extrahiert werden kann (keine Ziffern in der angegebenen Zeichenkette) - geben Sie {% if is_js %} null{% else %}Keine{% endif %} zurück. Lesen Sie die Vorbedingungen unten für einige zusätzliche Regeln.

Erwartetes Verhalten:

  • berücksichtige das LETZTE Zahlenzeichen vor der ERSTEN Ziffer;
  • berücksichtigen Sie den ERSTEN Punkt in der Zeichenkette (es ist garantiert, dass er nach mindestens einer Ziffer kommt oder mindestens eine Ziffer nach sich hat). Wenn ein Punkt vorhanden ist, wird als Float zurückgegeben, andernfalls - Integer.
